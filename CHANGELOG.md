# Changelog

## 1.0.0 - 2025-12-20
- Initial release of the Smoketree Swim & Recreation Club plugin.
- Added online registration with Stripe Checkout and manual payment fallback.
- Delivered member portal (profile management, guest passes, family/extra members, Stripe billing portal).
- Implemented guest pass purchasing, usage logging, and admin adjustments.
- Added admin dashboards, membership management UI, access codes, email campaigns, and CSV export.
- Created auto-renewal service with cron-based notifications and off-session Stripe payments.
- Introduced centralized logger, security hardening, rate limiting, and uninstall cleanup.
- Included 13 responsive HTML email templates and comprehensive documentation.
